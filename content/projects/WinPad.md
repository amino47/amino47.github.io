## WinPad Macropad
### What is it?
The WinPad is a 3 button macropad with QMK firmware that I designed.
### Why did I build it?
I built the WinPad for the Hack Club You-Ship-We-Ship Hackpad. The requirements of this challenge were to
1. Design a PCB for a macropad
2. Create firmware for the pad.
3. Create a 3D CAD model of a case for the pad.
If all these steps were completed, and the final project submitted, the nonprofit Hack Club would send you the case, PCB, and all parts neccesary to build the macropad. These would be sent to you free of charge, excluding customs fees.
